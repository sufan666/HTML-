# HTML全能编辑器

一个纯前端、零依赖的本地HTML可视化编辑工具。

## 特性

- **📂 打开/保存** — 加载本地 `.html` 文件，编辑后保存为独立文件
- **✏️ 可视化编辑** — 开启编辑模式， 直接点击文字即可修改内容，所见即所得
- **🎨 样式控制** — 调整背景色、文字色、宽高及最小尺寸
- **📏 拖拽缩放** — 选中元素右下角拖拽手柄，直观调整大小
- **↩️ 撤销/恢复** — Ctrl+Z / Ctrl+Y，所有操作皆可回溯
- **🔒 安全预览** — 默认禁用脚本，手动开启需确认信任

## 快速开始

1. 用浏览器打开 `HTMLEdit-Pro.html`
2. 点击「📂 打开」或拖拽 HTML 文件到预览区
3. 开启「编辑」开关，点击任意元素开始修改
4. 点击「💾 保存」下载编辑后的文件
